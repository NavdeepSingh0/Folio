## Encapsulation

**Definition:** Encapsulation is the bundling of data and methods that operate on that data into a single unit.

Encapsulation in C++ involves combining data members and member functions within a class, which restricts direct access to some components, thereby protecting the data from unintended modifications. This is achieved through access specifiers like private, protected, and public. Encapsulation also promotes data hiding and enhances the modularity and maintainability of the code.

> **Example:** A class representing a BankAccount with private data members for balance and methods for deposit and withdrawal.

> [!TIP]
> **Exam Focus:** Focus on how access specifiers control data visibility and how encapsulation improves code security and maintainability.
