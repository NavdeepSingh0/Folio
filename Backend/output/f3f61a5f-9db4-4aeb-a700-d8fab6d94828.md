## Floyd Warshall Algorithm

**Definition:** Floyd Warshall Algorithm computes the shortest path between every pair of vertices in a weighted graph using dynamic programming.

The algorithm solves the All Pairs Shortest Paths Problem by iteratively improving the shortest path estimates between all pairs of vertices. It works for graphs with positive and negative edge weights, as long as there are no negative weight cycles. The algorithm constructs a distance matrix where each element represents the shortest distance between two vertices. The dynamic programming approach updates the matrix by considering each vertex as an intermediate point, gradually refining the shortest paths. The initial matrix is populated with direct edge weights or infinity for non-edges. The algorithm's efficiency comes from its ability to compute all pairs in O(n^3) time, making it suitable for small to medium-sized graphs. Applications span network routing, airline optimization, and logistics.

**Algorithm Steps:**
1. Initialize the distance matrix with direct edge weights or infinity for non-edges.
2. Iterate through each vertex k as an intermediate node.
3. For each pair of vertices (i, j), check if the path through k provides a shorter distance.
4. Update the distance matrix with the minimum of the current value and the new path through k.
5. Repeat until all vertices have been considered as intermediate nodes.

```python
def floyd_warshall(graph):
    n = len(graph)
    dist = [[graph[i][j] for j in range(n)] for i in range(n)]
    for k in range(n):
        for i in range(n):
            for j in range(n):
                dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])
    return dist
```
